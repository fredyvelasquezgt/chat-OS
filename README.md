# Chat
Project 1 - OS


## Install

Please, be sure to install Ncurses and Protobuf in your environment before.

## Usage
```shell
make
```  
### Server
```shell
./server <port>
```  

### Client
```shell
./client <username> <server-ip> <server-port>
```
